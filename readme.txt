== Description ==

The plugin will assist you to increase the performance of one's WordPress website. We aim to make this happen by optimizing key highlights of the website’s loading process thus raising the loading swiftness and limiting the actual bandwidth consumption.

Feature:

1. Advanced caching system.
2. Database Optimization
3. Automatic Minify support for CSS, JavaScript and HTML.
4. Automatic gzip compression support
5. Expiry headers and eTag support.
6. Lazy Loader for smooth image loading.
7. Hotlink protection for your self hosted image to save bandwidth. 
8. Dedicated Administration Panel For Configuration of various settings. 
9. Almost ALL modern plugins and themes 
10. Control what to minify or not, what to cache or not and everything from the Option Page. 

== Installation ==

1. Upload Plugin folder to "WP-CONTENT/PLUGINS/" directory of your site.
2. Activate from the Admin Panel > Plugins. 
---------------------------------OR--------------------------------------
1. Navigate to Plugins > Add New.
2. Search For WP SpeedUp Performance Optimizer.
3. When found, click "Install Now". 
4. Activate! 
5. Visit Option Page for personalization of Settings.
